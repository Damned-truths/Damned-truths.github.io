Browser Support

Windows (and Linux mostly)

  • Blazer - press accesskey
  • Camino - press Ctrl + accesskey
  • Google Chrome 3 and higher - press Alt(Shift may be needed) + accesskey
  • Firefox 2 and higher - Alt + Shift + accesskey
  • Internet Explorer - Alt + accesskey
  • Konqueror - Ctrl +accesskey
  • Opera - Shift + Esc + accesskey
  • Safari 3 and higher - Alt + accesskey

Mac

  • Blazer - as for windows
  • Camino - as for windows
  • Google Chrome 3 and higher - Ctrl + Opt + accesskey
  • Firefox 2 and higher - Ctrl + accesskey
  • Internet Explorer - as for windows
  • Konqueror - as for windows
  • Opera - as for windows
  • Safari 3 - Ctrl + accesskey
  • Safari 4 and higher - Ctrl + Opt + accesskey
